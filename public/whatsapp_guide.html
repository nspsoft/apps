Cara, Setup & Konfigurasi

1. Pilih Provider Gateway

Masuk ke Settings > WhatsApp. Kami merekomendasikan Wablas atau Fonnte.

Rekomendasi: Wablas

2. Input API Token

Dapatkan API Token dari dashboard provider Anda. Paste ke kolom yang tersedia lalu klik Save.

3. Setup Webhook (PENTING!)

Salin URL Webhook dari halaman settings ERP, dan tempelkan di dashboard provider WhatsApp Anda.

https://jicos.jidoka.co.id/api/whatsapp/webhook

Halaman Konfigurasi

Settings > WhatsApp Center

Status Pesanan

Pelanggan cukup ketik nomor SO (contoh: SO-2023001) dan bot akan membalas status terkini (Processing, Shipped, Delivered) beserta info kurir.

Cek Invoice

Ketik Cek Tagihan untuk melihat daftar invoice yang belum dibayar. Bot otomatis menghitung total outstanding secara real-time.

Produk & Stok

Pencarian pintar. Ketik nama barang (misal: Pipa Besi), bot akan menampilkan harga dan stok terkini dari gudang.

Coba Simulator Bot

Klik tombol di bawah untuk melihat respon bot (Simulasi).
